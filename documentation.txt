Перед началом работы:
	1)создайте файл "base_data.json" и запоните его следующим текстом:
		{ 
			"1234": ['1'] 
		}
	2)скачайте нужные библиотеки:
		import aiogram
		import requests
		import beautifulsoup4
		import asyncio
	3) Получите токен у BotFather и введите его файл data_bot.py
	


Функию:
	</start> - Функция, которая приветсвтвует пользователя и выводит inline клавиатуру.
	<new> - Функия, которая проверяет появились новые аниме, после последней проверки.
	<search_message> - Функция, которая проверяет, есть ли новые серии пользоватоельского аниме.
	<scheduled_messages> - Функия, которая каждый день в 6, 12, 18 часов дня, запускает Функию search_message.
	<maill> - Функия, которая добавляет пользователя и его список аниме в базу данных.
