Перед началом работы:
	1)создайте файл "base_data.json" и запоните его следующим текстом:
		{ 
			"1234": ['1'] 
		}
	2)скачайте нужные библиотеки:
		import aiogram
		import requests
		import beautifulsoup4
		import asyncio
	3) Получите токен у BotFather и введите его файл data_bot.py
	


Функию:
	</start> - Функция, которая приветсвтвует пользователя и выводит inline клавиатуру.
	<new> - Функция, которая проверяет появились ли новые аниме, после последней проверки.
	<search_message> - Функция, которая проверяет, есть ли новые серии пользоватоельского аниме.
	<scheduled_messages> - Функция, которая каждый день в 6, 12, 18 часов дня, запускает функцию search_message.
	<maill> - Функция, которая добавляет пользователя и его список аниме в базу данных. После этого за пускает функцию scheduled_messages.
